1. 主板：华擎z390m-itx/ac；
2. cpu：9600k；
3. 硬盘：1T m2固态硬盘；
4. 内存条：海盗船16g x 2；

6. 显卡：使用核显uhd630，臭码农不配拥有独显；
7. Mac OS: 10.15-11.2.1 均可正常使用
